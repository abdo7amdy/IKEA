Project Name: IKEA Online Store Management System

Description:
A web-based e-commerce platform developed using ASP.NET Core MVC and Entity Framework Core, designed to simulate IKEAâ€™s online shopping experience. The system allows customers to browse products by categories (furniture, kitchen, office, etc.), view detailed product pages with descriptions and images, add items to a shopping cart, and complete secure checkout with payment simulation.

Key Features:

User authentication and role management (Admin & Customer)
Product catalog management (Add, Update, Delete products) for admins
Advanced product search and filtering
Shopping cart and order processing
Order history for customers
Responsive UI using Bootstrap
Database implemented with SQL Server and managed through EF Core
Technologies Used:
ASP.NET Core MVC, C#, Entity Framework Core, SQL Server, LINQ, Bootstrap, Razor Pages, and Identity
